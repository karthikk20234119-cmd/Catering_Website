# Royal Feast Catering

A premium, state-of-the-art catering services website built with React, TanStack Start, and Framer Motion. Featuring a cinematic UI/UX, responsive design, and luxury aesthetics.

## Features

- **Premium Design**: Dark mode with gold accents and smooth micro-animations.
- **Dynamic Content**: Signature menus, testimonial carousels, and interactive sections.
- **SEO Optimized**: Pre-configured meta tags for better search visibility.
- **Fast Performance**: Optimized assets and modern build pipeline.

## Tech Stack

- **Framework**: TanStack Start / React
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Deployment

### Vercel
1. Connect your GitHub repository to Vercel.
2. Ensure the **Framework Preset** is set to **TanStack Start**.
3. Ensure the **Output Directory** is set to `Default`.
4. Vercel will automatically build and deploy the project as a full-stack SSR application.
